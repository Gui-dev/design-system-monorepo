1.  Clone o repositório:

Bash

```
git clone https://github.com/Gui-dev/design-system-monorepo.git

```

3.  Instale as Dependências:

Bash

```
npm ci
```

3.  Execute a aplicação web:

Bash

```
npm run dev
```
